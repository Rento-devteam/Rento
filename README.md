# Rento - Монорепозиторий NestJS + React

## 🏗 Структура проекта
Rento/
├── packages/
│ ├── backend/ # NestJS backend
│ └── frontend/ # React + Vite frontend
## 🚀 Быстрый старт

### Установка
```bash
# Клонируйте репозиторий
git clone https://github.com/Rento-team/Rento.git
cd Rento

# Установите зависимости
npm install

Разработка
bash
# Запустить все проекты
npm run dev

# Или по отдельности
npm run dev:backend
npm run dev:frontend
📦 Команды
npm run dev - запустить все проекты в режиме разработки

npm run build - собрать все проекты

npm run lint - проверить код во всех проектах

npm run test - запустить тесты
