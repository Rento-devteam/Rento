    sequenceDiagram
    participant System as Система (Триггер)
    participant ARSEngine as Модуль расчета ARS
    participant Database as База данных
    participant ProfileService as Сервис профиля

    Note over System: Триггеры:<br/>- Верификация (UC-103)<br/>- Новый отзыв<br/>- Возврат в срок<br/>- Просрочка<br/>- Периодический пересчет

    System->>ARSEngine: trigger_ars_calculation(user_id, event_type)
    ARSEngine->>Database: Запрос данных пользователя
    
    Database-->>ARSEngine: Данные пользователя
    
    ARSEngine->>Database: Запрос наличия верификации
    Database-->>ARSEngine: verification_status (boolean)
    
    ARSEngine->>Database: Запрос отзывов
    Database-->>ARSEngine: reviews_data (оценки, количество)
    
    ARSEngine->>Database: Запрос статистики сделок
    Database-->>ARSEngine: transactions_stats<br/>(on_time_returns, overdue_returns)
    
    ARSEngine->>ARSEngine: Сбор параметров для расчета
    
    alt Недостаточно данных (новый пользователь)
        ARSEngine->>ARSEngine: Присвоение минимального/базового значения ARS
        Note over ARSEngine: Базовое значение = 50%<br/>или минимальный порог
    else Данные доступны
        ARSEngine->>ARSEngine: Расчет верификационного коэффициента
        Note over ARSEngine: Верификация: +20% к базе
        
        ARSEngine->>ARSEngine: Расчет рейтингового коэффициента
        Note over ARSEngine: Средняя оценка отзывов<br/>(вес 30%)
        
        ARSEngine->>ARSEngine: Расчет коэффициента надежности
        Note over ARSEngine: (on_time / total) * 100%<br/>(вес 50%)
        
        ARSEngine->>ARSEngine: Применение весовых коэффициентов
        Note over ARSEngine: ARS = (Verif * 0.2)<br/>+ (Rating * 0.3)<br/>+ (Reliability * 0.5)
    end
    
    ARSEngine->>Database: save_ars_value(user_id, ars_value, calculation_date)
    Database-->>ARSEngine: Значение сохранено
    
    ARSEngine->>ProfileService: update_profile_display(user_id, ars_value)
    ProfileService->>Database: Обновление кэша профиля
    
    alt Значение ARS изменилось значительно
        ProfileService->>System: trigger_notification(user_id, ars_changed)
        System->>System: Отправка уведомления пользователю
    end
    
    ProfileService-->>ARSEngine: Профиль обновлен
    ARSEngine-->>System: Расчет завершен
    
    rect rgb(240, 248, 255)
        Note over System,Database: Периодический пересчет по расписанию
        System->>ARSEngine: scheduled_batch_calculation()
        
        loop Для каждого активного пользователя
            ARSEngine->>Database: Получение списка пользователей
            Database-->>ARSEngine: user_ids
            ARSEngine->>ARSEngine: Расчет ARS для пользователя
            ARSEngine->>Database: Сохранение значения
        end
        
        ARSEngine-->>System: Пакетный расчет завершен
    end
