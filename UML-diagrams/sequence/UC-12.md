    sequenceDiagram
    actor User as Пользователь
    participant WebApp as Веб-приложение
    participant FavoritesService as Сервис избранного
    participant Database as База данных
    participant AuthService as Сервис аутентификации

    User->>WebApp: Нажатие на иконку "Сердечко"<br/>(на карточке вещи / в результатах поиска)
    
    WebApp->>AuthService: Проверка статуса авторизации
    
    alt Пользователь не авторизован
        AuthService-->>WebApp: unauthorized
        WebApp-->>User: Предложение авторизоваться/зарегистрироваться
        WebApp-->>User: Сообщение: "Войдите или зарегистрируйтесь, чтобы добавить в избранное"
        Note over WebApp,User: Опционально: сохранение намерения в localStorage
    else Пользователь авторизован
        AuthService-->>WebApp: user_id, authorized
        
        WebApp->>FavoritesService: GET /favorites/check?user_id={id}&listing_id={id}
        FavoritesService->>Database: Проверка наличия в избранном
        Database-->>FavoritesService: is_favorited (true/false)
        FavoritesService-->>WebApp: current_status
        
        alt Вещь НЕ в избранном (добавление)
            WebApp->>FavoritesService: POST /favorites (user_id, listing_id)
            FavoritesService->>Database: Создание связи user-listing
            Database-->>FavoritesService: связь сохранена
            FavoritesService-->>WebApp: 201 Created, status: added
            
            WebApp->>WebApp: Изменение иконки: пустое сердечко → активное
            WebApp-->>User: Визуальное подтверждение (сердечко закрашено)
            WebApp-->>User: Уведомление: "Добавлено в избранное"
            
        else Вещь В избранном (удаление)
            WebApp->>FavoritesService: DELETE /favorites?user_id={id}&listing_id={id}
            FavoritesService->>Database: Удаление связи user-listing
            Database-->>FavoritesService: связь удалена
            FavoritesService-->>WebApp: 200 OK, status: removed
            
            WebApp->>WebApp: Изменение иконки: активное сердечко → пустое
            WebApp-->>User: Визуальное подтверждение (сердечко пустое)
            WebApp-->>User: Уведомление: "Удалено из избранного"
        end
    end
    
    rect rgb(240, 248, 255)
        Note over User,Database: Просмотр избранного в профиле
        
        User->>WebApp: Переход в раздел "Избранное"
        WebApp->>FavoritesService: GET /favorites?user_id={id}
        FavoritesService->>Database: Запрос всех избранных вещей пользователя
        Database-->>FavoritesService: favorite_listings (список вещей)
        
        FavoritesService->>Database: Проверка актуальности каждой вещи
        Database-->>FavoritesService: listing_statuses (active/deleted/unavailable)
        
        FavoritesService-->>WebApp: favorites_data (с аннотацией о статусе)
        
        alt Есть недоступные вещи
            WebApp-->>User: Отображение списка избранных вещей<br/>Недоступные вещи отмечены как "Недоступна" / зачеркнуты
            WebApp-->>User: Уведомление: "Некоторые вещи больше недоступны"
        else Все вещи доступны
            WebApp-->>User: Отображение списка избранных вещей
        end
        
        loop Управление избранным
            User->>WebApp: Нажатие на сердечко (удаление из избранного)
            WebApp->>FavoritesService: DELETE /favorites (user_id, listing_id)
            FavoritesService->>Database: Удаление связи
            FavoritesService-->>WebApp: Удалено
            WebApp-->>User: Вещь удалена из списка
        end
    end
    
    rect rgb(255, 248, 240)
        Note over User,FavoritesService: Альтернативный сценарий: Массовые операции
        
        User->>WebApp: Выбор нескольких вещей в избранном
        User->>WebApp: Нажатие "Удалить выбранные"
        WebApp->>FavoritesService: DELETE /favorites/batch (user_id, listing_ids[])
        FavoritesService->>Database: Удаление всех указанных связей
        Database-->>FavoritesService: удалено N записей
        FavoritesService-->>WebApp: 200 OK (count_deleted)
        WebApp-->>User: Уведомление: "Удалено N вещей из избранного"
        WebApp-->>User: Обновленный список избранного
    end
    
    rect rgb(240, 255, 240)
        Note over WebApp,Database: Опционально: Аналитика для рекомендаций
        
        FavoritesService->>Database: Логирование события (для UC-11)
        Note over Database: Событие "добавление в избранное"<br/>используется AI-модулем<br/>для улучшения персонализации
    end
