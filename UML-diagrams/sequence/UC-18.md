    sequenceDiagram
    actor UserA as Пользователь (Отправитель)
    actor UserB as Пользователь (Получатель)
    participant WebApp as Веб-приложение
    participant ChatService as Сервис чата
    participant Database as База данных
    participant MediaService as Медиа сервис
    participant WebSocket as WebSocket сервер
    participant NotifyService as Сервис уведомлений
    participant ModerationService as Сервис модерации

    Note over UserA,UserB: Предусловия: Оба пользователя авторизованы, чат привязан к объявлению/сделке

    alt Инициализация чата
        UserA->>WebApp: Нажатие кнопки "Написать" на карточке вещи
        WebApp->>ChatService: GET /chat/initiate (listing_id, recipient_id)
        ChatService->>Database: Проверка существующего чата
        Database-->>ChatService: chat_exists / chat_id
        
        alt Чат не существует
            ChatService->>Database: Создание нового чата
            Database-->>ChatService: chat_id
            ChatService-->>WebApp: chat_id, chat_data
        else Чат существует
            ChatService-->>WebApp: chat_id, chat_data, message_history
        end
        
        WebApp-->>UserA: Отображение интерфейса чата
    end

    UserA->>WebApp: Ввод текста / Выбор изображения
    WebApp->>WebApp: Отображение локального превью (оптимистичное обновление)
    
    alt Отправка текстового сообщения
        UserA->>WebApp: Нажатие "Отправить"
        WebApp->>ChatService: POST /chat/{chat_id}/messages (text)
        
        ChatService->>ModerationService: check_message_content(text)
        ModerationService-->>ChatService: moderation_result
        
        alt Сообщение содержит запрещенный контент
            ChatService-->>WebApp: 403 Forbidden (content_blocked)
            WebApp-->>UserA: Уведомление: "Сообщение не отправлено. Обнаружен запрещенный контент"
        else Сообщение разрешено
            ChatService->>Database: Сохранение сообщения
            Database-->>ChatService: message_id, timestamp
            
            ChatService->>WebSocket: broadcast_message(chat_id, message_data)
            WebSocket-->>UserB: Доставка сообщения в реальном времени
            
            ChatService-->>WebApp: 200 OK (message_data)
            WebApp-->>UserA: Подтверждение отправки, сообщение в истории
            
            ChatService->>NotifyService: send_push_notification(recipient_id, chat_id, message_preview)
            NotifyService-->>UserB: Push-уведомление: "Новое сообщение от {sender}"
        end
        
    else Отправка изображения
        UserA->>WebApp: Нажатие иконки "Прикрепить файл"
        UserA->>WebApp: Выбор изображения на устройстве
        
        WebApp->>MediaService: POST /media/upload (file, chat_id)
        
        MediaService->>MediaService: Проверка формата и размера
        
        alt Неподходящий формат файла
            MediaService-->>WebApp: 415 Unsupported Media Type
            WebApp-->>UserA: Уведомление: "Поддерживаются только JPEG, PNG"
        else Превышен размер файла
            MediaService-->>WebApp: 413 Payload Too Large
            WebApp-->>UserA: Уведомление: "Файл превышает максимальный размер"
        else Файл валидный
            MediaService->>MediaService: Сжатие изображения (при необходимости)
            MediaService->>Database: Сохранение метаданных изображения
            Database-->>MediaService: media_id, url
            MediaService-->>WebApp: media_data (url, thumbnail_url)
            
            WebApp->>ChatService: POST /chat/{chat_id}/messages (image, media_id)
            ChatService->>Database: Сохранение сообщения с изображением
            Database-->>ChatService: message_id, timestamp
            
            ChatService->>WebSocket: broadcast_message(chat_id, message_data_with_image)
            WebSocket-->>UserB: Доставка изображения в реальном времени
            
            ChatService-->>WebApp: 200 OK (message_data)
            WebApp-->>UserA: Отображение превью изображения в чате
            
            ChatService->>NotifyService: send_push_notification(recipient_id, chat_id, "[Изображение]")
            NotifyService-->>UserB: Push-уведомление: "Новое сообщение от {sender}"
        end
    end
    
    rect rgb(240, 248, 255)
        Note over UserA,UserB: Просмотр истории сообщений
        
        UserA->>WebApp: Открытие существующего чата
        WebApp->>ChatService: GET /chat/{chat_id}/messages?limit=50&offset=0
        ChatService->>Database: Запрос истории сообщений
        Database-->>ChatService: message_history (текст, изображения, метаданные)
        ChatService-->>WebApp: history_data
        WebApp-->>UserA: Отображение истории чата (с пагинацией)
        
        loop Бесконечная прокрутка
            UserA->>WebApp: Прокрутка вверх
            WebApp->>ChatService: GET /chat/{chat_id}/messages?limit=50&offset=next
            ChatService->>Database: Запрос следующих сообщений
            Database-->>ChatService: older_messages
            ChatService-->>WebApp: older_messages_data
            WebApp-->>UserA: Добавление старых сообщений в историю
        end
    end
    
    rect rgb(255, 248, 240)
        Note over ModerationService,Database: Альтернативный сценарий: Пользователь заблокирован
        
        ChatService->>ModerationService: check_user_status(user_id, chat_id)
        ModerationService->>Database: Проверка статуса блокировки
        Database-->>ModerationService: user_status (active, blocked, read_only)
        
        alt Пользователь полностью заблокирован
            ChatService-->>WebApp: 403 Forbidden (account_blocked)
            WebApp-->>UserA: Уведомление: "Ваш аккаунт заблокирован. Чат недоступен"
        else Пользователь в режиме "только чтение"
            ChatService-->>WebApp: 200 OK (read_only=true)
            WebApp-->>UserA: Интерфейс чата доступен только для чтения
            WebApp-->>UserA: Уведомление: "Вы не можете отправлять сообщения. Ваш аккаунт ограничен"
        end
    end
    
    rect rgb(240, 255, 240)
        Note over WebApp,UserB: Индикация статуса
        
        WebSocket->>UserB: on_message_received
        UserB->>WebApp: Обновление интерфейса чата (если открыт)
        WebApp-->>UserB: Отображение нового сообщения / изображения
        
        alt Получатель не в чате
            NotifyService-->>UserB: Push-уведомление с превью
            UserB->>WebApp: Переход в чат
            WebApp->>ChatService: GET /chat/{chat_id}/messages?since=last_seen
            ChatService->>Database: Запрос непрочитанных сообщений
            Database-->>ChatService: unread_messages
            ChatService-->>WebApp: unread_data
            WebApp-->>UserB: Отображение новых сообщений с отметкой "Новые"
        end
        
        UserB->>WebApp: Просмотр сообщений
        WebApp->>ChatService: POST /chat/{chat_id}/read (last_read_timestamp)
        ChatService->>Database: Обновление статуса прочтения
        Database-->>ChatService: read_status_updated
        ChatService->>WebSocket: broadcast_read_receipt(chat_id, user_id)
        WebSocket-->>UserA: Индикация "Прочитано" в чате
    end
    
    rect rgb(255, 248, 240)
        Note over ChatService,Database: Сохранение метаданных чата
        
        Database->>Database: Сохранение:<br/>- chat_id, listing_id, deal_id (опционально)<br/>- participants (userA, userB)<br/>- created_at, updated_at<br/>- last_message_preview<br/>- unread_count для каждого участника
    end
