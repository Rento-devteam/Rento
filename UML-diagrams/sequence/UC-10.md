    sequenceDiagram
    actor User as Пользователь
    participant WebApp as Веб-приложение
    participant MapService as Картографический сервис
    participant SearchService as Сервис поиска
    participant Database as База данных
    participant GeolocationAPI as Geolocation API

    User->>WebApp: Переход на вкладку "Карта" / "Показать на карте"
    WebApp->>MapService: Инициализация карты
    MapService-->>WebApp: Карта загружена
    
    WebApp->>GeolocationAPI: Запрос геолокации
    
    alt Геолокация разрешена
        GeolocationAPI-->>WebApp: coordinates (lat, lng)
        WebApp->>MapService: setCenter(user_coordinates)
        MapService-->>WebApp: Центр установлен
    else Геолокация отклонена/недоступна
        WebApp->>MapService: setCenter(default_coordinates)
        MapService-->>WebApp: Центр установлен (по умолчанию)
        WebApp-->>User: Сообщение: "Геолокация отключена. Карта центрирована на Москве"
    end
    
    WebApp->>MapService: getCurrentBounds()
    MapService-->>WebApp: bounds (NE, SW)
    
    WebApp->>SearchService: GET /listings/map?bounds=NE,SW&zoom=level
    
    SearchService->>Database: Поиск вещей в видимой области<br/>(пространственный запрос)
    Database-->>SearchService: listings_in_bounds
    
    alt Вещи найдены в зоне видимости
        SearchService-->>WebApp: listings_data (coordinates, title, price, photo)
        WebApp->>MapService: addMarkers(listings)
        MapService-->>WebApp: Маркеры отображены
        
        WebApp-->>User: Отображение маркеров на карте
    else Вещи не найдены
        SearchService-->>WebApp: empty_result
        WebApp-->>User: Сообщение: "В данном районе пока нет предложений"
    end
    
    loop Динамическая загрузка при перемещении/масштабировании
        User->>WebApp: Перемещение карты / Изменение масштаба
        WebApp->>MapService: getCurrentBounds()
        MapService-->>WebApp: new_bounds
        
        WebApp->>SearchService: GET /listings/map?bounds=new_bounds&zoom=new_zoom
        
        alt Зона видимости изменилась
            SearchService->>Database: Пространственный запрос для новой области
            Database-->>SearchService: new_listings_in_bounds
            
            alt Новые вещи найдены
                SearchService-->>WebApp: new_listings_data
                WebApp->>MapService: clearMarkers()
                WebApp->>MapService: addMarkers(new_listings)
                MapService-->>WebApp: Маркеры обновлены
                WebApp-->>User: Отображение обновленных маркеров
            else Вещи не найдены
                WebApp->>MapService: clearMarkers()
                WebApp-->>User: Сообщение: "В данной области нет предложений"
            end
        end
    end
    
    User->>WebApp: Нажатие на маркер
    WebApp->>MapService: showInfoWindow(marker_id)
    MapService->>SearchService: GET /listings/{id}/preview
    SearchService-->>MapService: listing_preview (title, price, photo, rating)
    MapService-->>WebApp: Отображение всплывающего окна с краткой информацией
    WebApp-->>User: Отображение информации о вещи
    
    User->>WebApp: Нажатие на ссылку "Подробнее"
    WebApp->>User: Перенаправление на полную карточку вещи
    
    rect rgb(240, 248, 255)
        Note over User,MapService: Альтернативный сценарий: Поиск по адресу
        User->>WebApp: Ввод адреса в строку поиска на карте
        WebApp->>MapService: geocode(address)
        MapService-->>WebApp: coordinates
        WebApp->>MapService: setCenter(coordinates)
        WebApp->>SearchService: GET /listings/map?bounds=bounds_around_coords
        SearchService->>Database: Поиск вещей вокруг указанного адреса
        Database-->>SearchService: listings_near_address
        SearchService-->>WebApp: listings_data
        WebApp->>MapService: addMarkers(listings)
        WebApp-->>User: Отображение маркеров в указанном районе
    end
    
    rect rgb(255, 248, 240)
        Note over WebApp,Database: Оптимизация: Кластеризация маркеров
        WebApp->>MapService: enableClustering()
        Note over MapService: При большом количестве маркеров<br/>объединяются в кластеры<br/>с отображением количества
        MapService-->>WebApp: Кластеризованные маркеры
        WebApp-->>User: Отображение кластеров вместо отдельных маркеров
    end
